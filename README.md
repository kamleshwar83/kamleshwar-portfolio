# Kamleshwar Sharma – Graphic Design Portfolio

This is a complete HTML/CSS/JS document that serves as a professional portfolio and contact website for Kamleshwar Sharma, a Graphic Design Professional with over 12 years of experience.

## ✨ Features

- **Responsive Design** – Adapts to desktop, tablet, and mobile screens with a mobile-first approach and hamburger menu.
    
- **Modern UI** – Gradient accents, glassmorphism effects, animated blobs, scroll-triggered reveals, and smooth transitions.
    
- **Dark/Light Balanced Theme** – Navy, electric blue, gold, and off-white palette.
    
- **Floating Action Buttons** – WhatsApp chat, "Hire Me" quick link, and scroll-to-top.
    
- **Interactive Testimonial Slider** – Auto-rotating with dots, next/prev controls, and touch/swipe support.
    
- **Experience Timeline** – Expandable "Read More" sections for detailed work history.
    
- **Skills & Software Grids** – Visual cards for design expertise and tool proficiency.
    
- **PDF Portfolio Viewer** – Embedded iframe to display/download the complete portfolio PDF.
    
- **Contact Form** – Client-side validation + Formspree backend integration.
    
- **Mobile Drawer Navigation** – Off-canvas menu with focus trapping and accessibility enhancements.
